Responsive

This theme is responsive so it should work on most if not all devices with little to know work!

Some custom responsive styling was added to the responsive files. If you need to change how any of that behaves you can do so in the responsive less files. If you're not using Less, you can just modify in the css.

Built with Bootstrap and Less.

Formation was built with Bootstrap and less so it should be easy to update and customize.

Less files are included with the download. All customization was done in one custom.less file and you can change all your colors in the variables.less file.

Because all changes are done in the custom and variables file, you can still update bootstrap if you need/want to.
